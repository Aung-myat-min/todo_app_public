## Tech Stacks

- Flutter
- Firebase

## Getting Started

To run this app, following the instructions:

```
1. Install Flutter SDK and Andriod Studio
2. Create a Virtual Android Emulator (Minimum SDK - 23)
3. Run the emulator
4. run `flutter pub get; flutter run` in terminal
```
